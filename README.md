# EaglerUI
Easy to use frontend for selecting multiple versions of EaglerCraft. Recommended with use of [EaglerWeb](https://github.com/lax1dude/eaglerxserver)

## Features
- [x] Basic version selection

- [x] WASM toggle

- [x] Dark mode

- [ ] 1.5.2 support

- [ ] Other stuff

## How to use
1. Create a folder for each version you want to include on your webserver, for example if you want to add 1.12.2 then make a folder named `1.12.2`.
2. Create a folder named `wasm`, repeat step 1 inside that folder with WASM versions.
